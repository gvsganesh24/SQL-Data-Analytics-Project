# SQL Data Analytics Project

## Overview

This project demonstrates advanced SQL skills through exploratory data analysis (EDA), business reporting, and analytical problem-solving on sales data.

The project includes SQL scripts for database exploration, trend analysis, ranking, segmentation, cumulative metrics, performance analysis, and reporting.
